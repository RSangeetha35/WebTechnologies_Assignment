# 🎮 Guess the Number Game

## 📌 Project Description
The **Guess the Number Game** is a simple interactive web-based game developed using **HTML, CSS, and JavaScript**.  
In this game, the user tries to guess a randomly generated number between **1 and 10**.

The player starts with a score of **10 points**.  
For every incorrect guess, the score decreases by **1 point**.  
If the user guesses the correct number, they win the game. If the score reaches **0**, the game ends.

This project demonstrates basic concepts of **JavaScript logic, DOM manipulation, and user interaction** in web development.

---

## 🚀 Features
- Random number generation between **1 and 10**
- User input validation
- Score tracking system
- Instant feedback for correct or wrong guesses
- Alert messages for win or loss
- Simple and attractive UI using **CSS gradient design**
- Automatic game restart when score reaches **0**

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the web page  
- **CSS3** – Styling and UI design  
- **JavaScript** – Game logic and interaction  

---

## 🎯 How the Game Works
1. The user enters a number between **1 and 10**.
2. The system generates a **random number**.
3. If the guessed number matches the random number:
   - The user **wins the game**.
4. If the guess is wrong:
   - The **score decreases by 1**.
5. The game continues until:
   - The user guesses correctly, or  
   - The score becomes **0 (Game Over)**.
